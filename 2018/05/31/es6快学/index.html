es6快速预览

let&&const

var的问题:

  1. var没有作用域
  2. var不能定义常量
  3. var可以重复定义
    因为这些问题,es6引入了let和const来解决它们。es6之前js的作用域只有函数作用域和全局作用域。

  • let块级作用域

  • 不能提升作用域

  • 不会污染全局变量

  • 闭包实现

const 定义常量，常量一旦定义就不能修改

  • 定义值类型

  • 定义引用类型

解构

解构的意思就是分解一个对象,解构的时候，等号两边结构类似，右边还必须时以这真实的值。

  • 解构数组

  • 嵌套赋值

  • 省略赋值

  • 默认值(赋值和传参的时候可以使用默认值)

字符串

模板字符串

  • 模板字符串用反引号包含,其中的变量用${}括起来

  • 数组循环

  • 自己实现模板引擎

  • 带标签的模板字符串

可以在模板字符串的前面添加一个标签，这个标签可以去处理模板字符串,标签其实就是一个函数,因为我们有些时候希望有自己的拼接字符串逻辑,才有了这个标签的模板字符串.
rest运算符,会把后面的所有参数全都放在一个数组里,rest其他运算符只能作为最后一个参数

API
  • includes():返回布尔值，表示是否找到了参数字符串
  • startsWith()：返回布尔值，表示参数字符串是否在源字符串的头部。
  • endsWith()：返回布尔值，表示参数字符串是否在源字符串的尾部。
  • repeat:返回一个新字符串，表示将原字符串重复n次
